## ⚡ About The Project

**Rental** is a modern fullstack car rental web application where users can browse cars, check availability, and book vehicles easily.

This project is designed to simulate a real-world product with:
- Clean and modern UI/UX
- Booking system logic
- Admin dashboard
- Scalable backend structure

---

## 🚀 Features

### 👤 User Side
- 🚗 Browse available cars
- 🔍 Filter cars (price, brand, category, seats)
- 📅 Book cars by selecting dates
- ❤️ Add cars to favorites
- 📖 View booking history

### 🧑‍💼 Admin Panel
- ➕ Add / Edit / Delete cars
- 📊 Manage bookings
- 👥 Manage users
- 📈 Simple dashboard overview

---

## 🛠 Tech Stack

**Frontend:**
- React (Vite)
- Tailwind CSS
- React Router
- Axios

**Backend:**
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
